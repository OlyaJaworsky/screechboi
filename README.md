# screechboi

_A Reddit bot that performs sentiment analysis and screeches._

[![Drone][drone-img]][drone]

## Usage

```bash
# Clone repository.
git clone https://github.com/OlyaJaworsky/screechboi

# Enter project directory.
cd screechboi

# Run bot.
python .
```
